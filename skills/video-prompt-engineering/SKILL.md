# AI视频生成提示词工程 Skill

## 概述

本Skill整合了24个高质量提示词模板,涵盖AI视频生成的全流程:从小说改编、角色设计、分镜生成到最终视频提示词输出。支持多种AI视频工具(可灵、即梦、Vidu、NanoBananaPro等)。

## ⚠️ 核心原则: 提示词必须完整详尽

**不节省Token**: 视频生成提示词必须完整详尽，不得为节省Token而缩略删减。

**原因**: AI视频生成工具(可灵/即梦/Vidu等)依赖详细描述来理解场景、光影、动作和氛围。缩略提示词会严重影响视频质量，导致生成结果与预期不符。

**标准**:
- 单条视频提示词至少 **50-100中文字**
- 复杂场景需 **200+中文字**
- 必须包含: 主体描述 + 环境氛围 + 光影风格 + 镜头运动 + 质量标签
- **语言**: 使用**中文提示词**，AI视频工具对中文支持良好

**示例对比**:
```
❌ 缩略版 (15字) - 不可接受:
黑衣男子挥剑，金色闪电

✅ 完整版 (80字) - 标准:
古风玄幻黑衣少年侠客，身姿挺拔凌厉，手持长剑向前挥斩，剑身缠绕流动的金色闪电特效，
电光闪烁，火花四溅，45度侧逆光勾勒人物轮廓，背景是暗调玄幻秘境，冷蓝色调，
8K超高清，电影级光影，极致细腻渲染
```

## 核心工作流程

### 阶段1: 剧本改编 (可选)
```
输入: 网络小说原文
↓
[网文改编短剧提示词框架]
↓
输出: 剧本改编分析报告 + 场次规划
```

### 阶段2: 角色设计
```
输入: 角色描述/参考图
↓
[古装玄幻人物写真] + [Z-image艺术提示词]
↓
输出: 角色视觉档案 + 参考图提示词
```

### 阶段3: 分镜生成
```
输入: 剧本 + 角色设定
↓
[3x3分镜系统] 或 [漫剧九宫格] 或 [多人对话分镜]
↓
输出: 9-30条分镜(含时间轴/台词/声线)
```

### 阶段4: 视频生成
```
输入: 分镜描述
↓
[Seedance 2.0] 或 [Vidu图生视频] 或 [全能动作导演]
↓
输出: 可直接复制到AI视频工具的最终提示词
```

## 各模板详细说明

### 模板1: 网文改编短剧提示词框架
**用途**: 将网络小说改编为2-3分钟短剧剧本
**核心功能**:
- 剧本改编分析报告(7个维度)
- 场次规划(建议场次数)
- 角色档案动态更新
- 角色关系图(Mermaid语法)
- 剧本质量分析(7维度评分)

**输出格式**: Markdown表格 + 剧本格式

### 模板2: 古装玄幻人物写真
**用途**: 生成古风/玄幻风格的人物视觉提示词
**核心功能**:
- 角色视觉构建(肤色/五官/发型/服装)
- 光影哲学(3种光型)
- 材质真实化
- 强制校验准则

**输出格式**: 纯英文提示词(Next Scene格式)

### 模板3: Z-image艺术提示词(43组)
**用途**: 43组高质量艺术风格图像提示词
**涵盖风格**:
- 蒸汽朋克/赛博朋克
- 奇幻/超现实
- 电影级写实
- 抽象艺术
- 暗黑神话
- 动画/动漫

**输出格式**: Next Scene格式(中英文对照)

### 模板4: 3x3分镜提示词模板系统v5
**用途**: 专业级分镜生成(含台词声线)
**核心功能**:
- 角色设定系统(含声线)
- 违禁词自动替换
- 分阶段确认流程
- 15秒时长控制
- 9图=15秒映射

**输出格式**: 分镜表格 + 纯文本提示词

### 模板5: Seedance 2.0究极最终版
**用途**: 即梦网页端视频生成
**核心功能**:
- 一句话触发类型识别(A-H)
- 主体锁定(锚点)
- 降抽卡率硬规则
- 运镜与切镜控制
- 动作与力量感设计
- 节拍生成(每2秒节点)
- 台词与声音设计
- 字幕自动生成

**类型映射**:
A: 惊悚诡异 | B: 悬疑线索 | C: 剧情反转 | D: 搞笑反转
E: 高级动作 | F: 治愈情绪 | G: 氛围MV | H: 科普展示

**输出格式**: 即梦可直接粘贴的最终提示词

### 模板6: Vidu图生视频系统提示词
**用途**: Vidu平台动态视频生成
**核心功能**:
- 350字严格限制
- 音画同步描述
- 摄像机运动控制
- 专业运镜词汇
- 违禁词自动替换

**输出格式**: 纯文本描述(350字内)

### 模板7: 漫剧九宫格分镜
**用途**: NanoBananaPro 3x3宫格生成
**核心功能**:
- 视角锁定(人物ID不变)
- 特定镜头强化(背后/过肩/主观)
- 9分镜强制分布(2背后+3过肩+2主观+2随机)
- JSON格式输出

**输出格式**: JSON配置代码块

### 模板8: 全能动作导演Pro
**用途**: 15秒连贯动作戏份
**核心功能**:
- 四阶段动态设计(激战爆发/节奏博弈/巅峰对垒/终极余波)
- 物理反馈强制嵌入
- 电影级镜头调度
- 叙事连续性保障
- 600-1000字详细描述

**输出格式**: 长文本提示词(600-1000字)

### 模板9: 文生图智能体
**用途**: 通用视觉艺术提示词生成
**核心功能**:
- 黄金公式: 主体+环境+光影+相机+风格+色彩+质量
- 美学词库自动调用
- Midjourney参数自动添加

**输出格式**: 中文提示词 (如需英文可在末尾补充)

### 模板10: 多人对话分镜
**用途**: 复杂对话场景分镜
**核心功能**:
- 30条分镜输出
- 自我审查机制(17项检查)
- 字数严格控制(20-40字)
- 内心想法处理规则

**输出格式**: Markdown表格

## 统一输出规范

### 违禁词处理
所有模板统一执行以下替换:
- 鲜血→红色液体/深红色颜料
- 死亡→静止/沉睡/失去意识
- 杀戮→制服/阻止/对抗
- 暴力→冲突/推搡/拉扯

### 质量标签
所有输出必须包含:
- `"无时间码，无字幕"`
- `"8K超高清，极致细节，电影级质感，大师级作品"`
- `"8k, photorealistic, cinematic lighting"`

### 时长控制
- 短剧: 2-3分钟/集
- 分镜: 15秒/组(9图)
- Vidu视频: 4-15秒
- 提示词: 350字内(Vidu)或600-1000字(动作导演)
