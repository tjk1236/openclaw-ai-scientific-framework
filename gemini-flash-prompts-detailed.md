## 一、中文语义理解测试（古诗词意境）

### ZH-001 唐诗意境
**提示词：**
```
创作一幅中国古典山水画，画面中央是一条蜿蜒的河流，河面上有一艘小船，船上坐着一位身穿白色长袍的古代诗人，诗人抬头仰望天空。天空中挂着一轮明月，月光洒在河面上形成银色的波光。两岸是青翠的柳树，柳枝随风轻摆，远处是连绵起伏的山峦，山间有淡淡的云雾缭绕。整体采用水墨画风格，色调以黑白灰为主，点缀少许青色，营造出宁静、悠远、诗意的氛围。画面要有远近层次感，近景清晰，远景朦胧。
```
**测试点：** 理解"望明月"的意境，而非字面画一个"明亮的月亮"

### ZH-002 边塞豪情
**提示词：**
```
绘制一幅中国古代边塞场景，画面展现广袤的戈壁沙漠，黄沙漫天，地平线上可以看到雄伟的古城墙和烽火台。天空中夕阳西下，晚霞呈现出壮丽的橙红色。前景有几棵枯萎的胡杨树，树枝苍劲有力。远处有一队骑着马的士兵，旌旗飘扬，正在行军。画面要有苍凉、壮阔、豪迈的感觉，采用油画质感，色彩浓烈，对比强烈，表现出"大漠孤烟直，长河落日圆"的边塞诗意。
```
**测试点：** 理解边塞诗的苍凉壮阔，而非简单画沙漠

### ZH-003 田园隐居
**提示词：**
```
创作一幅田园隐居图，画面展现一个宁静的山村场景。近景是一座茅草屋，屋前有几棵高大的松树，树下有一位老者正在抚琴。屋旁是菜园，种着各种蔬菜。中景是一片金黄色的稻田，有几位农民正在劳作。远景是青翠的山峦，山间有瀑布飞流直下，山脚下有一条清澈的小溪。天空中有几朵白云，几只飞鸟掠过。整体色调温暖柔和，采用中国传统工笔画风格，细腻精致，表现出"采菊东篱下，悠然见南山"的闲适意境。
```
**测试点：** 理解陶渊明式隐居生活的细节

## 二、中文成语理解测试

### CH-001 画龙点睛
**提示词：**
```
画面展现一个中国古代画室场景。房间中央是一面巨大的白色墙壁，墙上已经画好了一条栩栩如生的巨龙，龙身蜿蜒，鳞片分明，龙爪锋利，但龙的眼睛位置是空白的。一位身穿唐代服饰的画师站在木梯上，手持毛笔，正准备给龙点睛。画师神情专注，笔触悬停在龙眼位置。画室中还有其他几幅已完成的作品，画案上摆放着砚台、墨块、颜料等绘画工具。窗外透进柔和的自然光，照亮了墙上的龙画。整个画面要有紧张期待的氛围，仿佛点睛之后龙就会飞走。采用写实风格，细节丰富。
```
**测试点：** 理解成语的核心是"点睛"这一关键动作，而非画一条龙

### CH-002 守株待兔
**提示词：**
```
画面展现中国古代农田场景。画面左侧是一棵粗壮的大树，树干上有一个明显的树桩断裂痕迹。一位身穿破烂农夫服装的中年男子正坐在树桩旁，背靠树干，双手抱在胸前，闭目养神，表情显得愚蠢而固执。他的农具（锄头、竹篓）被随意丢弃在身旁的田地里，田里的庄稼已经荒芜，长满了杂草。画面右侧远处可以看到一只兔子正在田边奔跑。天空中有几只飞鸟。整体色调偏黄昏时分，光线柔和但带有讽刺意味。画面要有寓言故事的教化感，农夫的懒惰和愚蠢要通过表情和姿态表现出来。采用类似中国古代绣像画的风格。
```
**测试点：** 理解寓言故事的讽刺意味，农夫的姿态是关键

### CH-003 对牛弹琴
**提示词：**
```
画面展现中国古代田园场景。画面中央是一位身穿长袍的文人雅士，正襟危坐，面前摆放着一张古琴，双手正在优雅地抚琴，神情专注而陶醉，仿佛正在演奏高深的乐曲。但画面的重点在于：他对面坐着一头大水牛，牛正在低头吃草，完全无视琴师的存在，甚至有一只苍蝇在牛耳边飞舞。背景是一片农田，远处有农舍和炊烟。画面要有强烈的反差感：一边是高雅的艺术，一边是愚昧的受众。采用略带夸张和幽默的漫画风格，突出"牛不懂音乐"这一核心笑点。
```
**测试点：** 理解"牛无视琴"这一关键笑点

## 三、网络流行文化理解

### NET-001 鸡你太美
**提示词：**
```
画面展现一个室内篮球场场景，地板是木质的，可以看到篮球架和观众席。画面中央是一位年轻男性，穿着黑色吊带背心搭配白色内搭，正在做篮球舞蹈动作，动作夸张搞笑。他的发型是中分，表情自信中带点滑稽。背景中可以看到一些模糊的观众，有的在拍照，表情是惊讶或好笑。画面整体色调明亮，采用类似表情包的风格，要有"梗文化"的幽默感。注意这不是在画一只鸡，而是表现"蔡徐坤篮球舞"这一网络梗的场景。
```
**测试点：** 理解"鸡你太美"指的是蔡徐坤篮球舞梗，不是字面画鸡

### NET-002 我太难了
**提示词：**
```
画面展现一个现代都市年轻人的生活场景。画面中央是一个二十多岁的年轻人，坐在电脑前，双手抱头，表情崩溃，眼睛下方有明显的黑眼圈，头发凌乱。电脑屏幕上显示着堆积如山的工作邮件或代码报错。桌面上散落着空咖啡杯、泡面盒、皱巴巴的纸张。窗外的天色已经暗下来，显示他加班到很晚。墙上贴着倒计时的日历，显示 deadline 临近。整体色调偏冷，光线昏暗，营造出压抑、焦虑、崩溃的氛围。采用写实风格，要能引起打工人的共鸣，表现出当代年轻人"压力山大"的生活状态。
```
**测试点：** 理解网络梗背后的情绪和生活状态

## 四、人物一致性测试（韩立IP）

### 基础形象描述
**韩立基础特征：**
- 性别年龄：青年男性，约20-25岁
- 面部：清秀俊朗，五官端正，眼神清澈坚定，眉毛细长
- 发型：黑色长发，用发簪束起，部分刘海垂落
- 服装：青色（青绿色）长袍，宽松飘逸，腰间系着玉带
- 配饰：腰间挂着储物袋，手腕可能有护腕
- 气质：修仙者气质，沉稳内敛，不张扬但自信

### CHAR-001 正面全身像
**提示词：**
```
创作一幅中国古风修仙人物全身像。画面中央是一位青年男性修仙者，名叫韩立。他站立姿势，双脚与肩同宽，双手自然垂在身侧。面部特征：清秀俊朗的五官，黑色长发用一根青色发簪束起，部分刘海自然垂落在额前，眼神清澈而坚定，表情沉稳内敛。身穿一件飘逸的青色长袍，长袍质地轻薄，随风微微飘动，腰间系着一条白色玉带，玉带上挂着一个小巧的储物袋。长袍下摆露出云纹靴子。背景是简单的纯色或淡淡的云雾，突出人物形象。整体采用中国传统工笔人物画风格，线条流畅，色彩清雅，光影柔和，表现出修仙者的超凡脱俗气质。高清细节，面部要清晰可辨。
```

### CHAR-002 侧面战斗姿态
**提示词：**
```
创作一幅中国古风修仙战斗场景。画面中的核心人物是韩立（青年男性修仙者，青色长袍，黑色长发束起，清秀面容，坚定眼神）。他正处于战斗姿态，身体侧向画面，双脚一前一后，呈弓步姿势。右手持剑向前刺出，剑身泛着淡淡的青光，剑尖有能量波动效果。左手捏着法诀，指尖有灵光闪烁。表情专注严肃，眼神锐利地盯着前方。青色长袍在战斗动作中剧烈飘动，衣褶要有动感。腰间玉带和储物袋清晰可见。背景是战斗场景，可以是天空或山巅，有能量爆炸或气流效果，但不能喧宾夺主。整体色调偏冷，以青蓝为主，光影对比强烈，突出战斗的紧张感。采用写实风格，人物面部要与正面像保持一致，确保是同一个人物。
```

### CHAR-003 打坐修炼
**提示词：**
```
创作一幅中国古风修仙修炼场景。画面中央是韩立（青年男性，青色长袍，黑色长发束起，清秀面容），他正以标准的打坐姿势坐在一个蒲团上，双腿盘起，双手放在膝盖上，掌心向上，呈修炼手印。双眼微闭，表情宁静平和，但眉宇间透露出一丝坚毅。周身有淡淡的青色灵光环绕，表示正在运转功法。头顶可能有灵气汇聚形成的小型漩涡。背景是一个简朴的洞府或静室，石壁上有淡淡的阵法纹路，角落摆放着一些简单的修仙器具（丹炉、药瓶、书籍等）。光线柔和，主要从上方照射下来，营造出神圣、静谧的修炼氛围。整体色调以青色和金色为主，画面要有仙气缭绕的感觉。采用细腻的中国画风，确保人物面部特征与之前一致。
```

### CHAR-004 远景山巅
**提示词：**
```
创作一幅中国古风山水画风格的远景场景。画面展现一座高耸入云的山峰之巅，云雾缭绕。在山顶的一块巨石上，站立着一个渺小的人物身影——韩立（虽然只是远景小身影，但要能看出青色长袍的轮廓和黑色长发）。他背对观众，面向远方的云海和初升的朝阳，长袍和长发在山顶狂风中猎猎作响。构图要大场面，人物占画面比例很小（约十分之一），重点表现壮阔的自然景观：层叠的山峦、翻涌的云海、金色的朝阳、飞翔的仙鹤。但人物虽小，姿态要清晰可辨：站立、望向远方、衣袍飘动。整体色调壮丽，从近景的深蓝到中景的紫红再到远景的金黄，形成渐变。采用中国传统山水画的大写意风格，但要保留人物的可识别特征（青色衣服、长发）。
```

### CHAR-005 近景面部特写
**提示词：**
```
创作一幅中国古风人物面部特写肖像。画面聚焦韩立的面部，从胸部以上取景，占满整个画面。面部细节要极其清晰：清秀俊朗的五官，剑眉星目，眼神清澈坚定中带着一丝历经沧桑的沉稳，高挺的鼻梁，薄唇紧闭，表情不怒自威但又不失温和。黑色长发有几缕垂在脸侧，背景虚化模糊。光照从侧面45度角打过来，在面部形成柔和的明暗对比，突出五官立体感。皮肤质感真实，能看到细微的毛孔和纹理，但不能有瑕疵。整体色调偏暖，采用超写实风格，类似高清摄影，但保留中国古典美男子的气质。这是用于面部识别对比的基准图，面部特征必须与其他变体保持一致。
```

### CHAR-006 夜间月光下
**提示词：**
```
创作一幅中国古风夜景人物图。画面中的韩立（青年男性，青色长袍，黑色长发束起，清秀面容）站在一片竹林中，时间是在夜晚，一轮满月高悬天空，清冷的月光透过竹叶洒落下来，在他身上形成斑驳的光影。他仰头望月的姿态，侧面轮廓被月光勾勒出一道银边。青色长袍在月光下呈现出偏蓝的色调，衣料的质感和褶皱清晰可见。表情略带忧郁和思念，眼神望向远方。周围是茂密的竹林，竹叶在夜风中轻轻摇曳。地面有淡淡的雾气。整体色调偏冷，以蓝银为主，营造出静谧、清冷、孤寂的月夜氛围。采用写实风格，人物面部特征必须与日间版本保持一致，只是光照环境不同。
```

## 五、九宫格专项测试提示词

### GRID-001 角色多角度展示（3x3）
**提示词：**
```
创作一个角色设计稿，采用3x3九宫格布局展示同一个角色的不同角度。角色是韩立（青年男性修仙者，青色长袍，黑色长发束起，清秀面容，坚定眼神）。九宫格的具体分布如下：
第一行（从左到右）：正面全身像、正面半身像、正面特写面部；
第二行：左侧全身像（90度侧面）、3/4侧面像（45度角）、右侧全身像（90度侧面）；
第三行：背面全身像、俯视角度、仰视角度。
所有九个格子中的角色必须是同一个人，面部特征、服装细节（青色长袍的样式、玉带的系法、储物袋的位置）、发型（束发方式、刘海形状）必须完全一致，只是观察角度不同。每个格子都要有简单的背景但不喧宾夺主。整体采用动漫人物设定稿的风格，线条清晰，色彩统一，方便用于3D建模参考。高清输出，每个格子都要清晰可辨。
```

### GRID-002 表情包九宫格（3x3）
**提示词：**
```
创作一个表情包集合，采用3x3九宫格布局，同一个角色（韩立，青年男性修仙者，青色长袍，黑色长发束起，清秀面容）展示九种不同的表情和反应。九宫格内容如下：
第一行：开心大笑（眼睛弯成月牙，嘴巴张开）、温柔微笑（嘴角上扬，眼神柔和）、无奈叹气（眉毛下垂，嘴角下撇）；
第二行：惊讶瞪眼（眼睛睁大，嘴巴微张）、愤怒生气（眉头紧皱，眼神凌厉）、委屈想哭（眼眶含泪，嘴角下撇）；
第三行：自信满满（昂首挺胸，嘴角上扬）、尴尬窘迫（眼神躲闪，挠头动作）、酷炫淡定（面无表情，眼神锐利）。
每个格子都要保持角色面部特征一致（五官位置、发型、脸型相同），只是表情变化。每个表情都要有适当的肢体动作配合。背景简单纯色。采用Q版或半Q版风格，适合用作聊天表情包。色彩鲜艳，表情夸张但可爱。
```

### GRID-003 动作序列九宫格（3x3）
**提示词：**
```
创作一个动作分解序列，采用3x3九宫格布局，展示韩立（青年男性修仙者，青色长袍）施展一套剑法的连续动作。九宫格按时间顺序从左到右、从上到下排列：
第一行：起势（双手握剑，剑尖向下，蓄力姿势）、拔剑（右手拔剑，剑身出鞘，青光初现）、上挑（剑身向上挑出，剑气形成月牙形）；
第二行：横斩（剑身水平横扫，剑气呈直线扩散）、旋身（身体旋转360度，剑随身转，形成圆形剑气）、下劈（双手持剑高举过头顶，向下劈砍，气势如虹）；
第三行：回防（剑身收回，格挡姿态）、突刺（剑尖向前直刺，速度感）、收势（剑收回鞘，气定神闲）。
九个动作必须是同一个角色的连续动作，服装、发型、面部特征保持一致，只是动作姿态变化。每个格子都要有动态模糊或速度线来表现动作感。背景统一为简单的演武场。采用动漫分镜风格，线条流畅，动作富有张力。适合用作动画原画参考。
```

### GRID-004 风格变体九宫格（3x3）
**提示词：**
```
创作一个风格对比图，采用3x3九宫格布局，同一个场景（韩立站在山巅，背对观众，望向云海日出，青色长袍飘动）用九种完全不同的艺术风格呈现：
第一行：日式动漫风格（大眼，线条简洁，色彩鲜艳）、美式漫画风格（粗黑线条，强烈阴影，高对比度）、皮克斯3D动画风格（圆润，材质感强，光影柔和）；
第二行：中国传统水墨画风格（黑白灰，写意，留白）、中国工笔画风格（细腻，色彩丰富，精致）、浮世绘风格（日本江户时代，平面化，线条装饰性）；
第三行：赛博朋克风格（霓虹灯，高科技，未来感）、像素艺术风格（8-bit，复古游戏风，方块感）、极简主义风格（几何形状，极少细节，纯色块）。
九个格子必须是同一个构图和场景，只是艺术风格不同。每个风格都要体现出该风格的典型特征，一眼就能分辨。整体采用对比展示的布局，方便 viewers 比较不同风格的表现力。
```

### GRID-005 色彩方案九宫格（3x3）
**提示词：**
```
创作一个色彩方案对比图，采用3x3九宫格布局，同一个画面（韩立站在桃花树下，青色长袍，春季场景）用九种不同的色彩方案呈现：
```

## 八、质量细节测试

### DETAIL-001 手部结构
**提示词：**
```
创作一幅写实风格的人物手部特写。画面聚焦于一双手，正在优雅地握住一只精致的陶瓷茶杯。手指纤细修长，指甲修剪整齐，皮肤纹理真实可见（能看到细微的毛孔和指纹，但不能有瑕疵或皱纹）。最重要的是：手指数量必须是五根，每根手指的关节结构要正确，不能有畸形或多余的手指。手指与茶杯的接触要有自然的阴影和压迫感，表现出握持的力度。茶杯是青花瓷，有精美的花纹。背景虚化模糊，突出手部主体。光线从侧面照射，在手部形成柔和的明暗对比。采用超写实风格，细节要极其清晰，适合用于检查AI画手的准确性。
```

### DETAIL-002 复杂场景透视
**提示词：**
```
创作一幅复杂的城市街景图，展现东京涩谷十字路口的繁忙景象。画面要有强烈的透视感，从 street level 视角看向十字路口中央。前景是熙熙攘攘的人群，至少有十几个人，各自在做不同的事情（看手机、走路、交谈），每个人都要有基本的形态特征，不能模糊成一团。中景是四条街道交汇的十字路口，能看到斑马线、交通信号灯、路牌。远景是高楼大厦，有巨大的电子广告牌（上面显示着日文广告和动漫角色），建筑风格要有东京特色。天空是黄昏时分，有晚霞。整体要有景深效果，近景清晰，远景适度虚化但不模糊。画面要表现出都市的繁华和喧嚣，但不能杂乱到无法分辨元素。采用写实风格，透视关系要正确。
```

### DETAIL-003 光影与反射
**提示词：**
```
创作一幅自然风景图，展现日落时分的湖面景象。画面中央是一片平静的湖泊，湖水清澈，倒映着天空和周围的景物。远处是连绵的山峦剪影，太阳正在山后落下，发出耀眼的金黄色光芒，光芒在湖面上形成一条金色的光路（glitter path），从太阳位置一直延伸到画面前景。天空被染成橙红、粉紫、金黄等渐变色。湖边有几棵树的剪影，树枝伸展。画面右侧可能有几只飞鸟。整体氛围宁静、壮丽、诗意。重点是光影的表现：太阳的 glow effect、天空的渐变、湖面的倒影、金色光路的 sparkle effect 都要真实自然。采用风景油画风格，色彩浓郁但不失真。
```

### DETAIL-004 材质质感
**提示词：**
```
创作一幅静物图，主体是一件红色的丝绒晚礼服。礼服挂在衣架上，展示出优雅的垂坠感和褶皱。重点是表现丝绒材质的独特质感：表面有细微的绒毛感，在光线照射下会有深浅不一的光泽变化（velvet特有的光泽是随着观察角度变化的），手感看起来柔软顺滑。礼服可能有精致的蕾丝装饰或珠片点缀。光线从侧面打过来，在丝绒表面形成柔和的高光和 shadow，突出材质立体感。背景是简单的暗色调（深灰或黑色），突出红色礼服。采用时尚摄影风格，质感要真实到能"感觉到"面料的触感。
```
