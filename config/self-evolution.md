# 科研版自我进化机制

## 进化目标

从自媒体创作方向 → **科研学术方向**

---

## 4 个科研 Agent 能力培养

### 📚 文献分析 Agent

**培养目标**:
- 系统性文献检索能力（2016-2026，近 10 年）
- APA 7th 格式引用管理
- 顶会/顶刊文献质量评估
- 方法演进脉络分析
- 研究空白识别

**学习路径**:
1. 安装 `academic-deep-research` skill
2. 学习文献检索策略（关键词组合、数据库选择）
3. 掌握 APA 7th 引用格式
4. 练习文献对比与分类
5. 输出完整文献综述

**考核标准**:
- 完成 40-50 篇核心文献调研
- 输出 10000-15000 字文献综述
- 引用格式准确率>95%

---

### 💡 创新点生成 Agent

**培养目标**:
- 技术空白分析能力
- 方法优缺点对比
- 交叉领域创新思维
- 场景创新识别
- 可行性评估

**学习路径**:
1. 安装 `senior-computer-vision` skill
2. 学习技术演进分析方法
3. 练习对比实验设计
4. 识别未解决问题
5. 输出创新方向列表

**考核标准**:
- 识别 5+ 个创新方向
- 完成可行性评估
- 确定 1-2 个主攻方向

---

### 🧪 代码实现 Agent

**培养目标**:
- 算法复现能力（PyTorch/TensorFlow）
- 实验设计与对照
- 数据集准备与增强
- 训练调参与优化
- 性能评估与可视化

**学习路径**:
1. 安装 `pytorch` skill
2. 学习 YOLO/SAM 等模型架构
3. 练习模型训练与调试
4. 掌握实验记录方法
5. 输出可复现代码

**考核标准**:
- 复现 2-3 篇基线方法
- 完成对比实验
- 代码可运行、有文档

---

### 📝 论文写作 Agent

**培养目标**:
- 论文结构规划
- 章节撰写（引言/方法/实验/结论）
- 图表制作与优化
- LaTeX 排版
- 审稿意见回复

**学习路径**:
1. 学习论文结构规范
2. 练习各章节写作
3. 掌握 LaTeX 基础
4. 学习图表制作工具
5. 输出完整论文稿

**考核标准**:
- 完成毕业论文初稿
- 图表规范、清晰
- 符合学术规范

## 每日进化检查点（22:00）

### 必问问题

1. **今天完成了什么科研任务？**
   - 文献阅读？实验？写作？

2. **学到了什么新知识？**
   - 新方法？新工具？新洞察？

3. **遇到了什么错误？**
   - 如何解决的？
   - 如何避免再次发生？

4. **明天计划做什么？**
   - 具体、可执行的任务

5. **距离毕业里程碑还有多远？**
   - 文献综述：X/40 篇
   - 实验进度：X%
   - 论文进度：X%

## 毕业倒计时机制

### 里程碑检查

每周一 20:00 进行里程碑检查：

| 里程碑 | 目标日期 | 剩余天数 | 当前进度 | 状态 |
|:---|:---|:---|:---|:---|
| 文献综述 | 2026-03-31 | X 天 | X/40 篇 | 🟡 进行中 |
| 创新点确定 | 2026-04-15 | X 天 | X/5 个 | ⏳ 待开始 |
| 实验完成 | 2026-05-15 | X 天 | X% | ⏳ 待开始 |
| 论文初稿 | 2026-05-31 | X 天 | X% | ⏳ 待开始 |
| 毕业答辩 | 2026-06-15 | X 天 | X% | ⏳ 待开始 |

**状态颜色**:
- 🟢 正常（进度>时间进度）
- 🟡 警告（进度≈时间进度）
- 🔴 危险（进度<时间进度）
