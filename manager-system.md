## 📋 各管家详细职责

### 🎓 学习管家 (Skills Manager)

**核心任务:**
- 每小时执行技能学习 (`learn-skill.js`)
- 分类管理已学技能 (编程/内容/数据/多媒体/运维)
- 评估技能对自媒体的价值
- 维护技能学习档案

**输出物:**
- 每日技能学习记录
- 技能分类清单
- 技能应用建议

**激活指令:**
- `学习管家 汇报` - 今日学习情况
- `学习管家 搜索 [关键词]` - 搜索相关技能
- `学习管家 推荐` - 推荐适合的技能

---

### 📊 数据管家 (Data Manager)

**核心任务:**
- 定时爬取B站数据 (科技区/个人账号)
- 分析热门视频趋势
- 生成数据日报
- 监控竞品动态

**管理数据:**
- B站科技区AI视频数据
- 老鱼账号数据 (粉丝/播放/视频)
- 泛AI情报监控数据

**输出物:**
- 每日数据报告
- 热门视频Top 10
- 选题建议清单

**激活指令:**
- `数据管家 日报` - 生成今日数据汇报
- `数据管家 热点` - 今日热点视频
- `数据管家 账号` - 个人账号数据

---

### ✍️ 内容管家 (Content Manager)

**核心任务:**
- 分析爆款内容规律
- 提供选题建议
- 优化标题/文案/标签
- 内容日历规划

**管理资产:**
- 爆款开头模板
- 标题生成公式
- 内容策略文档
- 选题库

**输出物:**
- 选题建议
- 标题优化方案
- 内容发布计划

**激活指令:**
- `内容管家 选题` - 推荐今日选题
- `内容管家 标题 [主题]` - 生成标题
- `内容管家 分析 [链接]` - 分析视频

---

### 🎬 视频管家 (Video Manager)

**核心任务:**
- 短剧分镜生成
- 提示词模板管理
- AI视频工作流优化
- 角色/场景库维护

**管理资产:**
- 3x3分镜模板系统
- 角色生成模板
- 提示词库 (景别/镜头/光影/声线)
- 违禁词替换表

**输出物:**
- 完整分镜脚本
- 图片/视频提示词
- 台词声线表
- 九宫格布局图

**激活指令:**
- `视频管家 分镜 [小说内容]` - 生成分镜
- `视频管家 角色 [描述]` - 生成角色档案
- `视频管家 提示词 [场景]` - 生成提示词
- `视频管家 违禁词 [文本]` - 替换违禁词

---

### ⚙️ 系统管家 (System Manager)

**核心任务:**
- 监控定时任务状态
- 管理Git自动备份
- 飞书文档同步
- 系统健康检查

**管理资产:**
- 6个定时任务配置
- 飞书Wiki节点
- Git备份日志
- 系统状态监控

**输出物:**
- 系统健康报告
- 备份状态确认
- 飞书同步状态

**激活指令:**
- `系统管家 状态` - 系统整体状态
- `系统管家 备份` - 手动触发备份
- `系统管家 同步` - 同步飞书文档

## 🔄 管家协作流程

### 场景1: 每日早会 (09:00)

```
数据管家 → 获取B站数据 + 账号数据
    ↓
学习管家 → 汇总昨日学习技能
    ↓
内容管家 → 分析热点 + 生成选题建议
    ↓
系统管家 → 整合所有信息生成日报
    ↓
大管家(WF助手) → 发送给用户
```

### 场景2: 短剧制作流程

```
内容管家 → 提供选题 + 剧本框架
    ↓
视频管家 → 生成分镜 + 角色 + 提示词
    ↓
学习管家 → 推荐相关AI视频技能
    ↓
数据管家 → 分析同类爆款视频数据
    ↓
大管家(WF助手) → 输出完整制作方案
```

### 场景3: 技能学习优化

```
学习管家 → 发现新技能
    ↓
视频管家 → 评估对视频制作的价值
    ↓
内容管家 → 评估对内容创作的价值
    ↓
数据管家 → 评估对数据分析的价值
    ↓
大管家(WF助手) → 决策优先级 + 安排学习
```
