## 二、刚学到的专业脚本结构（从参考脚本分析）

### 1. 完整脚本组成要素

| 要素 | 内容 | 作用 |
|------|------|------|
| **标题库** | 6大类标题（主爆款/新手引流/白嫖福利/垂直精准/短钩子/SEO） | 覆盖不同流量入口 |
| **时长规划** | 每环节精确到分钟（0:00-1:00开场/1:00-2:30认知...） | 把控视频节奏 |
| **画面指导** | 每个环节配画面说明（"循环播放8张截图"/"切PPT思维导图"） | 拍摄/剪辑可直接执行 |
| **口播话术** | 完整台词，可直接念 | 降低UP主准备成本 |
| **BGM建议** | 快节奏/放缓等提示 | 控制情绪节奏 |
| **踩坑提示** | 预判新手常见问题 | 提高完播率和收藏率 |
| **数据背书** | 反复强调真实成果（824粉/3.3%爆款率） | 增强可信度 |

### 2. B站知识区爆款结构（14-15分钟）

```
【0:00-1:00】黄金开场：成果炸场+直击痛点
【1:00-2:30】认知铺垫：概念科普+价值说明
【2:30-4:30】核心选型：方案对比+推荐
【4:30-8:00】核心干货：保姆级教程
【8:00-10:00】必学环节：关键配置演示
【10:00-12:30】进阶教学：高阶玩法
【12:30-13:30】福利环节：资源分享
【13:30-14:30】结尾总结+互动引导
```

### 3. 标题策略（6大类型）

1. **实战成果前置** - 用数据背书（日涨800粉）
2. **新手零门槛** - 戳痛点（零代码/复制粘贴）
3. **白嫖福利向** - 强引导互动（免费领/送模板）
4. **垂直受众** - 分人群（博主/打工人/技术党）
5. **短平快钩子** - 信息流抓眼球（5分钟搞定）
6. **SEO搜索向** - 埋关键词（2026最新教程）

### 4. 话术技巧

- **开场**: 成果展示→痛点共鸣→承诺价值
- **教学**: 步骤编号→命令展示→强调极简
- **过渡**: "接下来就是..."/"这里给大家..."/"很多新手..."
- **结尾**: 总结回顾→三连引导→资源领取→账号关注

## 四、SKILL功能规划

### 核心功能

**输入**:
- 视频主题/工具名称
- 目标受众（新手/进阶/垂直领域）
- 视频时长（5分钟/15分钟/30分钟）
- UP主风格（数据型/幽默型/硬核型）
- 个人成果数据（用于背书）

**输出**:
- 6类标题备选
- 完整分镜脚本（含时长/画面/台词）
- 关键步骤命令/代码
- 踩坑提示清单
- 封面/缩略图建议

### 差异化特色

- **B站特化**: 针对B站算法优化（完播率/互动率）
- **知识区专用**: 适配AI/技术/教程类内容
- **数据驱动**: 自动插入用户提供的成果数据
- **多风格**: 支持不同UP主人设（幽默/严谨/亲和）
